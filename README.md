# DLLTWZ_GUI
This is a team project for the real time class.


##一、项目名称
###**桌面冰球游戏GUI子系统**

##二、设计目标

 - 输入：击球器、冰球的当前位置和速度。 
 
 - 输出：每个控制周期两个驱动轴的实时位置和速度指令。

 - 主要参数：与物理引擎子系统的参数相一致。
 
 - 功能：能实时显示击球器和冰球在游戏桌上的运动，并提供适当的人机操作接口。
